# Alumni Giving Rate Analysis     
A linear regression analysis identifying factors that influence alumni donation rates, applied to university fundraising data.   

**Overview**  
This project applies statistical modeling techniques to understand what drives alumni giving behavior. The analysis focuses on identifying significant predictors of donation rates and interpreting their practical implications for university development strategy.

**Methods**   
- Simple and multiple linear regression  
- Model assumption testing (linearity, normality, homoscedasticity)   
- Predictor significance testing and interpretation   
- Model diagnostics and residual analysis   

**Key Questions**   
- Which alumni characteristics most strongly predict donation rates?   
- How do factors like graduation year, degree type, and class size relate to giving behavior?   
- Based on our findings, where should fundraising efforts be focused?   

**Tools**  
R · Linear Regression · Statistical Modeling · R Markdown   
**Output**   
Analysis is available as both an R Markdown source file (.Rmd) and a formatted Word document report.
